# IPL_Analysis
End-to-end IPL Data Analysis project using SQL, Python, and Power BI to analyze team performance, player statistics, season insights, and interactive dashboards.

# 🏏 IPL Analysis Dashboard | SQL + Python + Power BI

## 📌 Project Overview

This project provides a complete analysis of the Indian Premier League (IPL) using SQL, Python, and Power BI. It transforms raw IPL match data into meaningful business insights through data cleaning, exploratory data analysis, SQL querying, and interactive dashboards.

The project demonstrates a complete Data Analytics workflow from data extraction to dashboard visualization.

---

## 🎯 Business Problem

The IPL generates a massive amount of match data every season. This project answers important business questions such as:

- Which team has won the most matches?
- Which team has the highest winning percentage?
- Who are the top-performing players?
- Which stadium hosts the most matches?
- How does toss decision affect match results?
- Which teams perform better while batting first or chasing?
- Who won each IPL season?
